🗄️ Exploring Financial Market Data using SQL

As part of my data analytics learning journey, I worked with stock market data of Ashok Leyland using SQL in DBeaver with a SQLite database.

I executed SQL queries to retrieve and explore trading data such as Open, High, Low, Previous Close, Last Traded Price (LTP), and Closing Price for different dates.

🔎 What this analysis involved:

Creating and connecting to a database using SQLite

Importing stock market dataset into a table

Writing SQL queries to retrieve financial trading records

Exploring daily market indicators such as Open, High, Low, and Close prices

📊 Why this is important:
Working with SQL allows analysts to efficiently store, query, and manage large financial datasets, making it easier to perform further analysis in tools like Python or business intelligence platforms.

💡 Key takeaway:
Combining SQL with financial datasets helps build strong foundations in data management, financial analytics, and structured data querying.

This project helped strengthen my skills in SQL querying, database management, and financial data exploration.
